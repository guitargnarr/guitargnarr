## Recent Work (Dec 2025)

**Guitar Platform Pipeline** - Full stack system for generating Guitar Pro files:
- Deterministic music theory engine (12 scales, 8 patterns, 4 tunings)
- FastAPI backend deployed on Render
- React frontend with GP5 export button
- 2,415 test combinations, 100% accuracy

**Why Python over LLM**: Tested Ollama models for tab generation - they consistently produce musically incorrect output (F# in E Phrygian). Deterministic Python = 100% accuracy.
